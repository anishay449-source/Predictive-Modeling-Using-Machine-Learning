# Predictive-Modeling-Using-Machine-Learning :::
Machine Learning project using Random Forest Classifier to predict student performance based on study hours and attendance with accuracy evaluation and confusion matrix visualization.


# Project Overview
This project demonstrates the implementation of a Machine Learning model using the Random Forest Classifier algorithm to predict student performance based on study hours and attendance. The model is trained and tested on a dataset and evaluated using accuracy score and a confusion matrix.

# Features
* Data loading and preprocessing using Pandas
* Feature and target selection
* Train-test data splitting
* Random Forest Classifier implementation
* Model training and prediction
* Accuracy evaluation
* Confusion Matrix visualization
* Performance analysis using Scikit-Learn

# Technologies Used
* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-Learn

# Dataset
The dataset contains:
* Hours_Studied
* Attendance
* Passed (Target Variable)
Where:
* 0 = Fail
* 1 = Pass

# Machine Learning Workflow ::
# 1. Load Dataset
The dataset is loaded using Pandas.

# 2. Feature Selection
Hours Studied and Attendance are selected as input features.

# 3. Target Selection
Passed column is selected as the target variable.

# 4. Train-Test Split
The dataset is divided into training and testing sets using an 80:20 ratio.

# 5. Model Training
A Random Forest Classifier is trained using the training data.

# 6. Prediction
The trained model predicts student outcomes on the test dataset.

# 7. Model Evaluation
Accuracy Score is used to evaluate the model performance.

# 8. Confusion Matrix
A confusion matrix is generated and visualized to analyze prediction results.

# Output
* Accuracy Score
* Confusion Matrix Visualization
